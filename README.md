# 📈 Zerodha Clone

A full-stack stock trading platform inspired by Zerodha, built using the MERN Stack. This project includes landing page, user authentication, a trading dashboard, holdings management, and stock visualization.

## 🚀 Live Demo

### Frontend
https://zerodha-clone-five-omega.vercel.app

### Dashboard
https://zerodha-dashboard-black.vercel.app

### Backend API
https://zerodha-backend-3fi6.onrender.com

---

## ✨ Features

- 🔐 User Signup & Login
- 📊 Interactive Holdings Dashboard
- 📈 Stock Holdings Graph
- 💼 Portfolio Management
- 🛒 Buy/Sell Order Management
- ☁️ Deployed on Vercel & Render
- 🗄️ MongoDB Atlas Database

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/anuragchaurasia143/zerodha-clone.git
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
npm start
```

Dashboard

```bash
cd dashboard
npm install
npm start
```

Backend

```bash
cd backend
npm install
npm start
```
